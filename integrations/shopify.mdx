## Activate Chat Widget on Website

After installation, enable the chat widget on your storefront:

<Steps>
  <Step title="Navigate to Online Store">
    In Shopify admin: **Sales Channels** ‚Üí **Online Store**
    
    <Frame>
      <img src="/images/integrations/shopify-online-store-menu.png" alt="Online Store menu" />
    </Frame>
    
    *Screenshot: Shopify sidebar showing Sales Channels ‚Üí Online Store*
  </Step>
  
  <Step title="Go to Themes">
    Click **Themes** ‚Üí **Customize**
    
    <Frame>

    </Frame>
    
    *Screenshot: Themes page with "Customize" button*
  </Step>
  
  <Step title="Open App Embeds">

## Troubleshooting

<AccordionGroup>
  <Accordion title="Widget not appearing on store">
    **Check:**
    1. App embed is toggled to **Active**
    2. Theme customizer changes were **Saved**
    3. Clear browser cache and refresh
    4. Check theme compatibility
    5. View published theme (not preview)
  </Accordion>

  <Accordion title="Chat not responding">
    **Check:**
    1. Agent is deployed (`lua production`)
    2. Skills are pushed (`lua push`)
    3. Check browser console for errors
    4. Verify agent ID is correct
    5. Test in admin dashboard
  </Accordion>

  <Accordion title="Can't find app embeds">
    **Location:**
    - Theme Customizer ‚Üí Left sidebar ‚Üí App embeds
    
    **If missing:**
    - Ensure app is installed
    - Try different theme
    - Contact Shopify support
  </Accordion>

  <Accordion title="Products not showing in chat">
    **Check:**
    1. Products are published in Shopify
    2. Products have inventory
    3. Agent has product search tools
    4. Test with simple product names
  </Accordion>
</AccordionGroup>
