## Welcome to Lua CLI

Lua CLI is a powerful command-line tool for building, testing, and deploying AI skills. A **skill** is a collection of **tools** (functions) that your AI agent can use to accomplish tasks.

## Quick Example

```typescript
import { LuaTool, LuaSkill } from 'lua-cli';
import { z } from 'zod';

// Define a tool
class GetWeatherTool implements LuaTool {
  name = "get_weather";
  description = "Get current weather for a city";
  
  inputSchema = z.object({
    city: z.string()
  });

  async execute(input: { city: string }) {
    // Call weather API
    return {
      temperature: 72,
      condition: "sunny",
      city: input.city
    };
  }
}

// Create a skill
const weatherSkill = new LuaSkill({
  name: "weather-skill",
  version: "1.0.0",
  description: "Provides weather information",
  context: "Use get_weather when users ask about weather",
  tools: [new GetWeatherTool()]
});
```

## Next Steps

<Steps>
  <Step title="Install Lua CLI">
    Install the CLI globally with npm
    ```bash
    npm install -g lua-cli
    ```
  </Step>
  <Step title="Authenticate">
    Set up your API key
    ```bash
    lua auth configure
    ```
  </Step>
  <Step title="Create Your First Skill">
    Initialize a new project
    ```bash
    mkdir my-skill && cd my-skill
    lua init
    ```
  </Step>
  <Step title="Start Development">
    Test your skill with live reload
    ```bash
    lua dev
    ```
  </Step>
</Steps>
