## How It Works

<Steps>
  <Step title="Write Tools (TypeScript Functions)">
    Create functions that integrate with your APIs or external services
    
    ```typescript
    class OrderLookupTool implements LuaTool {
      async execute(input) {
        // Call YOUR API
        const response = await fetch('https://your-api.com/orders/' + input.orderId);
        return await response.json();
      }
    }
    ```
  </Step>
  
  <Step title="Bundle Tools Into Skills">
    Group related tools together
    
    ```typescript
    const customerServiceSkill = new LuaSkill({
      name: "customer-service",
      tools: [new OrderLookupTool(), new CreateTicketTool()]
    });
    ```
  </Step>
  
  <Step title="Deploy Your Agent">
    One command to push your agent live
    
    ```bash
    lua push && lua deploy
    ```
  </Step>
  
  <Step title="Users Chat, AI Takes Actions">
    Users chat naturally, AI calls your tools to take real actions
    
    **User**: "What's the status of my order #12345?"
    
    **AI**: *Calls your OrderLookupTool* → Returns real order status
  </Step>
</Steps>

## What You Can Build

### Integrate With YOUR Backend

Connect to your existing systems:

```typescript
// Call YOUR API endpoints
class MyCustomTool implements LuaTool {
  async execute(input) {
    // YOUR authentication
    const apiKey = env('YOUR_API_KEY');
    
    // YOUR endpoints
    const response = await fetch('https://your-backend.com/api/endpoint', {
      headers: { 'Authorization': `Bearer ${apiKey}` }
    });
    
    // YOUR business logic
    const data = await response.json();
    return transformYourData(data);
  }
}
```

## Example: Three Different Approaches

### Approach 1: Your Own APIs Only

```typescript
import { LuaTool, LuaSkill } from 'lua-cli';
import { z } from 'zod';

// Connect to YOUR existing backend
class GetCustomerTool implements LuaTool {
  name = "get_customer";
  description = "Look up customer in your CRM";
  inputSchema = z.object({ email: z.string().email() });

  async execute(input: { email: string }) {
    // Call YOUR API
    const response = await fetch(`https://your-crm.com/api/customers/${input.email}`, {
      headers: { 'X-API-Key': env('YOUR_CRM_API_KEY') }
    });
    
    return await response.json();
  }
}

const skill = new LuaSkill({
  name: "crm-integration",
  description: "Access our CRM system",
  tools: [new GetCustomerTool()]
});
```

### Approach 2: Platform APIs Only

```typescript
import { LuaTool, Products, Baskets } from 'lua-cli';

// Use Lua's built-in e-commerce APIs
class ShopAssistantTool implements LuaTool {
  name = "add_to_cart";
  description = "Add product to cart";
  
  async execute(input: { productId: string }) {
    // Use Platform APIs (optional convenience)
    const product = await Products.getById(input.productId);
    const basket = await Baskets.create({ currency: 'USD' });
    await Baskets.addItem(basket.id, {
      id: input.productId,
      price: product.price,
      quantity: 1
    });
    
    return { basketId: basket.id };
  }
}
```

### Approach 3: Mix Both (Most Common)

```typescript
import { LuaTool, Data } from 'lua-cli';

// Combine YOUR APIs + Platform APIs
class SmartSearchTool implements LuaTool {
  name = "smart_search";
  description = "Search across all your systems";
  
  async execute(input: { query: string }) {
    // 1. Search YOUR product API
    const yourProducts = await fetch('https://your-api.com/search', {
      method: 'POST',
      body: JSON.stringify({ query: input.query })
    }).then(r => r.json());
    
    // 2. Also search Lua's vector search (for FAQs, docs, etc.)
    const faqResults = await Data.search('faqs', input.query, 5, 0.7);
    
    // 3. Combine results
    return {
      products: yourProducts,
      faqs: faqResults.data,
      message: `Found ${yourProducts.length} products and ${faqResults.count} FAQs`
    };
  }
}
```

### Complete Solution (Hybrid)

```typescript
import { Data, env } from 'lua-cli';

class CompleteSearchTool implements LuaTool {
  async execute(input: { query: string }) {
    // 1. Search YOUR product database
    const myProducts = await fetch(`https://your-db.com/search?q=${input.query}`)
      .then(r => r.json());
```

## Quick Start Examples

### Option 1: Pure Custom Integration

No Platform APIs - just your own integrations:

```typescript
import { LuaTool, LuaSkill, env } from 'lua-cli';
import { z } from 'zod';

class YourCustomTool implements LuaTool {
  name = "your_tool";
  description = "Does something specific to your business";
  inputSchema = z.object({ param: z.string() });

  async execute(input) {
    // Call YOUR API, YOUR database, YOUR services
    const apiKey = env('YOUR_API_KEY');
    const response = await fetch('https://your-api.com/endpoint', {
      headers: { 'Authorization': `Bearer ${apiKey}` }
    });
    
    return await response.json();
  }
}

const skill = new LuaSkill({
  name: "your-business-skill",
  description: "Custom integration with your systems",
  tools: [new YourCustomTool()]
});
```

### Option 2: Use Platform APIs

Quick start with built-in e-commerce:

```typescript
import { LuaTool, Products, Baskets } from 'lua-cli';

class QuickShopTool implements LuaTool {
  name = "shop";
  description = "Browse and buy products";
  
  async execute(input: { query: string }) {
    // Use Platform APIs for quick e-commerce
    const products = await Products.search(input.query);
    const basket = await Baskets.create({ currency: 'USD' });
    
    return { products, basketId: basket.id };
  }
}
```

### Option 3: Best of Both Worlds

Mix your integrations with Platform helpers:

```typescript
import { LuaTool, Data, env } from 'lua-cli';

class HybridSearchTool implements LuaTool {
  async execute(input: { query: string }) {
    // 1. Search YOUR Shopify store
    const shopifyProducts = await fetch(
      `https://your-store.myshopify.com/admin/api/2024-01/products.json?query=${input.query}`,
      { headers: { 'X-Shopify-Access-Token': env('SHOPIFY_TOKEN') }}
    ).then(r => r.json());
    
    // 2. Use Platform API for knowledge base (vector search)
    const helpArticles = await Data.search('help', input.query, 5);
    
    return {
      shopifyProducts: shopifyProducts.products,
      helpArticles: helpArticles.data
    };
  }
}
```
