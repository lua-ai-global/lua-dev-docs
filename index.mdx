## How It Works

<Steps>
  <Step title="Write Tools (TypeScript Functions)">
    Create functions that integrate with your APIs or external services
    
    ```typescript
    class OrderLookupTool implements LuaTool {
      async execute(input) {
        // Call YOUR API
        const response = await fetch('https://your-api.com/orders/' + input.orderId);
        return await response.json();
      }
    }
    ```
  </Step>
  
  <Step title="Bundle Tools Into Skills">
    Group related tools together
    
    ```typescript
    const customerServiceSkill = new LuaSkill({
      name: "customer-service",
      tools: [new OrderLookupTool(), new CreateTicketTool()]
    });
    ```
  </Step>
  
  <Step title="Configure Your Agent">
    Use LuaAgent to configure everything in one place
    
    ```typescript
    export const agent = new LuaAgent({
      name: "support-agent",
      persona: "You are a helpful customer support agent...",
      skills: [customerServiceSkill]
    });
    ```
  </Step>
  
  <Step title="Deploy Your Agent">
    One command to push your agent live
    
    ```bash
    lua push && lua deploy
    ```
  </Step>
  
  <Step title="Users Chat, AI Takes Actions">
    Users chat naturally, AI calls your tools to take real actions
    
    **User**: "What's the status of my order #12345?"
    
    **AI**: *Calls your OrderLookupTool* â†’ Returns real order status
  </Step>
</Steps>

## Three Ways to Integrate

<Tabs>
  <Tab title="Your APIs Only">
    **100% Custom Integration**
    
    ```typescript
    // Connect to YOUR systems
    class YourTool implements LuaTool {
      async execute(input) {
        const response = await fetch('https://your-backend.com/api/endpoint', {
          headers: { 'X-API-Key': env('YOUR_API_KEY') }
        });
        return processYourData(await response.json());
      }
    }
    ```
    
    Perfect for existing infrastructure, custom business requirements, or any industry
  </Tab>

  <Tab title="Platform APIs">
    **Optional Built-In Helpers**
    
    ```typescript
    import { Products, Baskets, Data } from 'lua-cli';
    
    // Quick e-commerce or data storage
    const products = await Products.search('laptop');
    const basket = await Baskets.create({ currency: 'USD' });
    const results = await Data.search('docs', 'query');
    ```
    
    Great for rapid prototyping, e-commerce MVPs, or when you don't have a backend yet
  </Tab>

  <Tab title="Mix Both">
    **Best of Both Worlds**
    
    ```typescript
    import { Data } from 'lua-cli';
    
    class HybridTool implements LuaTool {
      async execute(input) {
        // YOUR Shopify store
        const products = await fetch('https://your-store.myshopify.com/api/...');
        
        // Platform vector search for FAQs
        const faqs = await Data.search('help', input.query);
        
        return { products, faqs };
      }
    }
    ```
    
    Most flexible approach - use the best tool for each job
  </Tab>
</Tabs>

<Note>
**Platform APIs are completely optional!** They're convenience helpers for common e-commerce and data tasks. You can build agents that integrate with ANY system using standard HTTP/REST/GraphQL calls.
</Note>

## Simple Example

```typescript
import { LuaAgent, LuaSkill, LuaTool } from 'lua-cli';
import { z } from 'zod';

// 1. Create a tool that calls ANY external API
class GetWeatherTool implements LuaTool {
  name = "get_weather";
  description = "Get current weather for a city";
  
  inputSchema = z.object({
    city: z.string()
  });

  async execute(input: { city: string }) {
    const response = await fetch(
      `https://api.openweathermap.org/data/2.5/weather?q=${input.city}`
    );
    
    const data = await response.json();
    
    return {
      temperature: data.main.temp,
      condition: data.weather[0].description,
      city: input.city
    };
  }
}

// 2. Bundle into a skill
const weatherSkill = new LuaSkill({
  name: "weather-skill",
  description: "Provides weather information",
  tools: [new GetWeatherTool()]
});

// 3. Configure your agent (v3.0.0)
export const agent = new LuaAgent({
  name: "weather-assistant",
  persona: "You are a friendly weather assistant. Provide accurate weather information for any city.",
  welcomeMessage: "Hi! Ask me about the weather anywhere in the world.",
  skills: [weatherSkill]
});
```

Deploy with: `lua push && lua deploy`
