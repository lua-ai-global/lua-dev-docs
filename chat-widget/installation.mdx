## CDN Installation (Recommended)

Perfect for static websites, WordPress, Shopify, or any HTML page.

###Step 1: Add Script Tag

Add this code right before the closing `</body>` tag:

```html
<script src="https://lua-ai-global.github.io/lua-pop/lua-pop.umd.js"></script>
```

### Step 2: Initialize

```html
<script>
  window.LuaPop.init({
    agentId: "your-agent-id",
    position: "bottom-right",
    theme: "light"
  });
</script>
```

### Complete Example

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">

      position: "bottom-right",
      theme: "light",
      buttonText: "Chat with AI"
    });
  </script>
</body>
</html>
```

## Single Page Applications

### React

```tsx
// components/ChatWidget.tsx
import { useEffect } from 'react';

export default function ChatWidget() {
  useEffect(() => {
    // Load LuaPop script
    const script = document.createElement('script');
    script.src = 'https://lua-ai-global.github.io/lua-pop/lua-pop.umd.js';
    script.onload = () => {
      window.LuaPop?.init({
        agentId: process.env.NEXT_PUBLIC_AGENT_ID,
        position: "bottom-right",
        theme: "light",
        sessionId: `user-${Date.now()}`
      });
    };
    document.body.appendChild(script);

    // Cleanup
    return () => {
      document.body.removeChild(script);
    };
  }, []);

  return null;
}

// In your main App component
import ChatWidget from './components/ChatWidget';

function App() {
  return (
    <div>
      {/* Your app */}
      <ChatWidget />
    </div>
  );
}
```

### Next.js

```tsx
export default function RootLayout({ children }) {

              position: "bottom-right",
              theme: "light"
            });

    </html>
  );
}
```

## Security Considerations

<Warning>
**Never expose sensitive agent credentials in client-side code!**
</Warning>

### Best Practices

- ‚úÖ Use environment-specific agent IDs
- ‚úÖ Use `staging` environment for development
- ‚úÖ Generate unique session IDs per user
- ‚úÖ Don't store user PII in `runtimeContext`
- ‚úÖ Use HTTPS on your website

### Example Secure Setup

```javascript
window.LuaPop.init({
  agentId: process.env.LUAPOP_AGENT_ID, // From environment
  environment: process.env.NODE_ENV === 'production' ? 'production' : 'staging',
  sessionId: generateSecureSessionId(), // Your session logic
  authToken: await getAuthToken() // If using authenticated users
});
```
